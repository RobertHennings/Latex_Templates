\chapter{Overview}
This design example uses the compact submission title page. It demonstrates a
lighter front page followed by a complete but short thesis-like document.

\section{Design Intent}
The submission layout emphasizes essential metadata and leaves institutional
branding more restrained than the classic cover.

\chapter{Analysis}
\section{Narrative Text}
A paragraph of sample text is useful for checking margins, line spacing, and
font sizes across the body pages. The template should remain readable for both
short and long documents.

\section{Enumerated Items}
\begin{enumerate}
  \item Prepare the metadata.
  \item Select a title-page variant.
  \item Add the thesis content.
  \item Compile twice for stable references.
\end{enumerate}

The minimal submission example provides a second full-document preview with a
different cover setting.
