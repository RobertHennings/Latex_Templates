\chapter{Introduction}
This compact design example shows the classic CAU thesis title page together
with representative thesis content. It is intended for visual inspection of
headings, paragraphs, lists, links, and bibliography output.

\section{Motivation}
A reusable thesis template should provide a stable structure while leaving room
for project-specific content. The sample text keeps the document short but
exercises the most common layout elements.

\section{Example List}
\begin{itemize}
  \item Metadata is configured before \verb|\begin{document}|.
  \item The title page is imported from \verb|cau_thesis/title_pages|.
  \item Body content starts after the front matter.
\end{itemize}
