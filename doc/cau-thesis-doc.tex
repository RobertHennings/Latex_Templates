\chapter{Overview}
\texttt{cau\_thesis} is a reusable thesis class based on LaTeX's standard
\texttt{report} class. It provides thesis metadata commands, a generic title
page, link styling, bibliography helpers, theorem environments,
algorithm/listing support, and project-compatible title-page hooks.

\chapter{Minimal Example}
\begin{lstlisting}[language=TeX]

\chapter{Metadata Commands}
The class uses setter/getter commands for thesis metadata. Calling a command
with an argument stores the value; calling it without an argument prints the
stored value where supported by existing title pages.

\begin{lstlisting}[language=TeX]
\thesisauthor{<name>}
\stunumber{<number>}
\stuemail{<email>}
\privatemail{<email>}
\submissiondate{DD.MM.YYYY}
\submissiondateCover{<text>}
\pdfsubject{<text>}
\pdfkeywords{<comma-separated keywords>}
\chair{<text>}
\institute{<text>}
\faculty{<text>}
\typedocument{<text>}
\programme{<text>}
\cooperation{<text>}
\location{<text>}
\license{<label>}{<url>}
\licencse{<label>}{<url>}
\githubrepo{<label>}{<url>}
\GitHubRepo{<label>}{<url>}
\end{lstlisting}

The misspelled \verb|\licencse| command is retained as a deprecated compatibility alias.

\chapter{Title Pages}
Use \verb|\makethesistitlepage| for a generic title page that does not require
private institutional images. Project-specific title-page files can still be
included manually when their assets are available and legally redistributable.

\chapter{Link Options}
\begin{lstlisting}[language=TeX]
\linkboxcolor{<color>}
\showlinkboxes{true}
\showlinkboxes{false}
\frontmatterpageanchor{true}
\hypersetup{pageanchor=true}
\end{lstlisting}

\verb|\linkboxcolor| changes visible link border colors. The class starts with
\verb|pageanchor=false| for front matter; re-enable anchors after title pages
with \verb|\hypersetup{pageanchor=true}| when needed.
